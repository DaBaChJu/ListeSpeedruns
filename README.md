Programme fait pour gérer une liste de vidéos de type speedrun.
Récupère la durée des vidéos depuis un fichier externe MediaInfo.dll.
Permet le tri selon le nom de fichier, la durée ou le numéro (ordre d'ajout).

Program to manage a list of videos of the speedrun type.
Get videos duration from an external file MediaInfo.dll.
Allow to sort by filename, duration or number (based on add time).
